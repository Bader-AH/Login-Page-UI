# Login-Page-UI
[https://bader-ah.github.io/Login-Page-UI/](https://bader-ah.github.io/Login-Page-UI/)

A modern and professional Login Page interface built as part of my web development journey. This project focuses on clean design, user experience (UX), and responsive layout using core web technologies.

## 🚀 Live Demo
You can view the live version of this project here: 
[https://bader-ah.github.io/Login-Page-UI/](https://bader-ah.github.io/Login-Page-UI/)

## ✨ Key Features
* **Modern Aesthetic:** Clean and minimalist design with high-quality focus.
* **Gradient Button:** Features a stylish purple-to-red gradient for a modern feel.
* **Responsive Input Fields:** Clearly defined username and password fields with professional placeholders.
* **Semantic HTML:** Built with clean and readable HTML5 and CSS3 code.

## 🛠️ Technologies Used
* **HTML5:** For the structural layout of the page.
* **CSS3:** For styling, including layouts, fonts, and button gradients.
